Gestion_actividades v1.0.0-alpha

Corrección de vista para alumnos:
- El bloque visible en el curso ya no enlaza a la pantalla interna de ediciones.
- Ahora enlaza a workshop_view.php, que es la página pública del taller.
- El bloque visible muestra:
  * nombre,
  * descripción,
  * fecha,
  * horas,
  * plazas,
  * botón “Ver taller / inscribirme”.
- Si ya existía una etiqueta antigua con “Abrir ediciones del taller”, al pulsar “Generar talleres en el curso” se actualiza en vez de duplicarse.

Importante:
- Las pantallas de ediciones siguen existiendo para gestión, pero no son el destino del alumno.
