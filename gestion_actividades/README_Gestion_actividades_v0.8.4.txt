Gestion_actividades v0.8.4-alpha

Cambio de usabilidad:
- En el listado de talleres se elimina el icono +.
- El icono de lápiz ya no abre los datos base del taller.
- El lápiz abre la edición completa:
  * si hay una edición, abre esa edición;
  * si no hay ninguna, abre la creación de la primera edición;
  * si hay varias, abre la lista de ediciones para elegir.
- La lupa se mantiene para ver todas las ediciones.

Objetivo:
- Evitar confusión entre editar taller base y configurar la edición real del taller.
