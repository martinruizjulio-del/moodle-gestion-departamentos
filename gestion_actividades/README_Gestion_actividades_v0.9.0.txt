Gestion_actividades v0.9.0-alpha

Nueva función:
- Se puede borrar un taller desde la pantalla principal de Talleres.
- Se añade icono de papelera en la columna Acciones.
- Antes de borrar aparece pantalla de confirmación.

Al borrar un taller se intenta eliminar:
- el taller base,
- sus ediciones,
- profesores vinculados a las ediciones,
- inscripciones internas,
- registros de horas del taller,
- entrada visual del curso creada como etiqueta,
- grupos Moodle asociados a ediciones, si existieran.

Nota:
- No se borran automáticamente actividades complejas de Moodle salvo la etiqueta visual creada por el plugin para mostrar el taller en la sección TALLERES TIPO A.
