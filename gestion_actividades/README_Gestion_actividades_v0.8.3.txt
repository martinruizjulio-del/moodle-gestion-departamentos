Gestion_actividades v0.8.3-alpha

Correcciones de flujo y secciones:
1) Aclara la diferencia entre taller base y edición del taller.
- Editar taller base solo muestra curso, nombre, horas y descripción.
- Fecha, plazas, profesores, tarea/cuestionario, grupo, asistencia y certificado se configuran en "Nueva edición de taller" o editando una edición.

2) Crea secciones del curso al guardar el taller:
- TALLERES TIPO A
- TALLERES TIPO A - ARCHIVO
- Una sección propia del taller: CÓDIGO - Nombre del taller

3) La tarea/cuestionario automática se colocará en la sección propia del taller.

4) Se añaden mensajes de ayuda para evitar que el gestor piense que los datos de edición faltan al editar el taller base.

Nota:
- Si antes se creó una sección llamada "Nueva sección", puede ser una sección previa sin nombre. Al guardar de nuevo el taller con esta versión, se crearán las secciones con nombre correcto.
