Gestion_actividades v0.7.1-alpha

Corrección conceptual:
- El alumno NO genera el certificado.
- La pantalla ya no debe inducir a pensar que el alumno puede generar el certificado.
- Se cambia el enfoque a:
  * Cumple requisitos para certificado.
  * El sistema/gestor deberá registrar y almacenar el certificado.
  * El alumno solo lo consultará o descargará una vez esté almacenado.

Flujo confirmado:
1) Alumno inscrito en la edición del taller.
2) Profesor registra asistencia.
3) Alumno completa la tarea/cuestionario asociado.
4) El alumno cumple requisitos de certificado.
5) En la siguiente fase, el sistema registrará/almacenará el certificado mediante Custom Certificate / My Certificates / portafolio.

Esta versión NO implementa todavía el almacenamiento real del certificado; solo corrige textos y prepara el flujo para esa integración.
